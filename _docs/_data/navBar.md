# **[API](#API)**

参数|类型|默认值|说明
-|-|-|-
type|String|normal|类型，可选值"none","base","normal"。
color|String|-|颜色，可选值"none","success","info","warning","error","dark","calm","stable","light"。为空时根据type自行判断, none为强制清除颜色
sideButton|String|both|侧边按钮，可选值"none","left","right","both"
